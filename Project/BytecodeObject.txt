Bytecode Object
Xi Scripting Language
Febuary 2013

--- Overview ---
	A Xi Bytecode Object is a compiled binary object that contains all the bytecode generated by the
Xi compiler. It allows for faster execution of precompiled code and for some cross-platform execution.

--- Change Log ---
Version 1 [Febuary 27th, 2013]:
	- Initial version

--- Format ---
Version		: 1
Revision Date	: Febuary 27th, 2013
Magic		: XBCO

--- Global Format ---
<Header>
<Count>
For <Count> {
	<Module>
}
<End of File>

--- Header ---
 0 4 Magic
 4 2 Version
 6 2 Compiler Version
 8 2 Minimum VM Version

--- Module ---
 0 2 Length		{ String <Name> }
 0 2 Field Count

 <Method>								[ Body Method ]

 2 4 Count
 For <Count> {
	<Class>
 }

 0 4 Count
 For <Count> {
	<Method>
 }

--- Class ---
 0 2 Length		{ String <Name> }
 0 2 Length		{ String <Base Class Name> }
 2 2 Field Count
 4 4 Count
 For <Count> {
	<Method>
 }

--- Method ---
 0 2 Length		{ String <Name> }
 0 2 Argument Count
 2 2 Variable Count
 4 4 Length		{ Stream <Instructions> }
